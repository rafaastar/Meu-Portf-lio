Jogo do Botão Fujão

Um divertido jogo onde o botão tenta escapar do clique do jogador. O objetivo é capturar o botão fujão antes que ele fuja novamente.

Replit Github

Jogo de Quiz

Criamos um jogo de perguntas e respostas em que o jogador deveria escolher a resposta correta para avançar no jogo. Utilizamos HTML, CSS e JavaScript para desenvolver a interface e a lógica do jogo, permitindo ao usuário ver seu progresso e receber feedback instantâneo sobre suas respostas.

Replit GitHub

Mercadinho Virtual

Nesse projeto, simulamos uma loja online simples, onde o usuário podia adicionar produtos ao carrinho e ver o valor total da compra. Utilizamos JavaScript para atualizar os valores em tempo real e CSS para criar um design atraente e funcional.

Replit GitHub

Jogo Fruit Ninja

Inspirado no famoso jogo mobile, criamos uma versão simplificada onde frutas apareciam na tela e o jogador deveria "cortar" as frutas ao clicar nelas. Foi um projeto divertido que envolveu a criação de animações, efeitos de som e pontuação.

Replit GitHub

Carrossel Animado

Desenvolvemos um carrossel de imagens dinâmico, onde as imagens trocavam automaticamente ou podiam ser controladas pelo usuário. Este projeto nos permitiu trabalhar com transições de CSS e eventos de clique, além de oferecer uma experiência visual interativa.
